# Grandma's Food
## Home-cooked goodness, in a flash.

Here you will find the repository for the `grandmas-food` microservice project. 

To run the infrastructure, do a `docker compose up` on the project's directory!

## Ports
For development purposes, the following ports are assigned to each of the services:

| Service        |   Port   |
|:---------------|:--------:|
| Reverse Proxy: |   `80`   |
| API Gateway    |  `8080`  |
| Auth           |  `8081`  |
| Products       |  `8082`  |
| Shopping Cart  |  `8083`  |
| Payment        |  `8084`  |
